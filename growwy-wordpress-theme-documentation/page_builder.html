Page Builder Introduction

Page Builder Introduction

This theme is created with Popular Elementor Page Builder. Theme comes with over 50 custom modules as well. Its easy and fun to customise the website to your requirements.

Find out more about that plugin here https://elementor.com/

Here is quick start guide...

  • To create/edit a page.. Log in as admin and Go to Pages ( add new or edit ) and then click Edit with Elementor. Or you can also click Edit with Elementor from Top admin bar from front page if you are logged in as admin.(as shown below)

Note: If you need, Documentation for the plugin can be found here https://elementor.com/support/
