Setup other pages

Setup Other Pages.

Here are the steps to do so:

  1. Log in to Hubspot, from the top panel, go to Marketing > Website > Website Pages.

  2. From the top right corner button, click Create > Website page

  3. On next page, select the Ranbron theme.

  4. On next page, select the appropriate template for your page, for example, for About page, choose About.

  5. Give this new page a name, eg About.

  6. Click Create page button.

Once you finish above steps, the Page will load with default modules, and populated with demo content, you can then point, click and edit the content to suit your needs. You can also drag/drop other modules as needed from the left panel, as well as you can delete the unneeded modules by hovering them on the page editor.
